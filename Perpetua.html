Perpetua
This page is under construction. We are currently looking for images to add to it. Please DM ev on Discord if you have any images that may be worthwhile. Thank you!
Perpetua are a rare currency used to perform Evocations. They are a permanent currency; their name is derived from the word perpetual. They are the rarest and most difficult to obtain currency, and are sometimes called “perps”.

Creating Perpetua consumes Ephemera Levels, making them an even more exclusive currency than Ephemera. For more information on the Evocations they can perform, refer to the sections below.
Perpetua are only obtained through using the command !crystallize This reduces the user's Ephemera Level by 16 and creates one Perpetua. Users can hold at most 13 Perpetua each. If 89 Perpetua exist at once, the Collapse event will be triggered, setting all users' credits, Ephemera, Ephemera Levels, Perpetua, and Pieces to 0. This does not affect Potentia.

As the cost of Ephemera Levels constantly fluctuates, there is no fixed number of credits required to create a Perpetua. However, it is possible to estimate such a cost based on Active Credits:

$$\text{Cost}=0.002\cdot\text{Active credits}\cdot\sum_{i=1}^{16}i=0.272\cdot\text{Active credits}$$

It is also possible to calculate the cost of a Perpetua in Potentia as the value of Potentia are also tied to the number of Active Credits:

$$\text{Number of Potentia}=\cfrac{0.272\cdot\text{Active credits}}{0.0005\cdot\text{Active credits}}=544$$

And since each Potentia requires $1 USD to purchase, it follows that the value of a single Perpetua is $544 USD.
The main use of Perpetua is to perform Evocations, which are extremely powerful commands that can affect many aspects of TaylorBowl. In the command !perpetua, the description of Evocations is hidden if the command user has no Perpetua. Holding at least 1 Perpetua will reveal all Evocations.
Disintegrate
Disintegrate is used to instantly gain Ephemera. The command !disintegrate consumes 1 Perpetua to grant the evoker 32 Ephemera, while also destroying all other Ephemera in existence. This ignores the evoker's Ephemera limit.
Forget
Forget is used to remove all votes from a Bowl. The command !forget consumes 1 Perpetua to remove all votes for both tracks in the current Bowl. The effects of Forget ignore Fortify and Reinforce.
Annihilate
Annihilate is used to instantly force The Blight. The command !annihilate consumes 1 Perpetua to cause the Blight to occur immediately with no Warning Bloom, even if the Threshold is below 100%. The true name for the Warning Bloom can be found within the text for Annihilate in the command !perpetua.
Submerge
Submerge is used to reduce all Ephemera Levels. The command !submerge consumes 1 Perpetua to reduce the Ephemera Level of all users by 1. This cannot reduce a user's Ephemera Level below 0.
Listen
Listen is used to allow the evoker to monitor Ephemera and Perpetua activities. The command !listen consumes 1 Perpetua to make BowlBot send a direct message to the evoker each time a user uses an Invocation or Evocation until the end of the Season.
Turn
Turn is used to remove all votes for one track in a Bowl. The command !turn consumes 2 Perpetua to remove all voters from the track option the evoker has not voted in. It is unknown what will happen if the evoker has not yet voted. The effects of Turn ignore Fortify and Reinforce.
Convert
Convert is used to switch all votes in a Bowl. The command !convert consumers 3 Perpetua causes all voters in a given Bowl to vote for the track the evoker has voted for. The effects of Convert ignore Fortify and Reinforce.
Pariah
Pariah is used to Disintegrate other Perpetua. The command !pariah consumes 13 Perpetua to cause up to 13 other Perpetua to Disintegrate, yielding half the regular amount of Ephemera to the owners of said Perpetua.
Martyr
Martyr is used to distribute Perpetua. The command !martyr consumes 13 Perpetua to cause 13 Perpetua to be distributed to random holders of Perpetua. If there are no other holders of Perpetua, the Evocation misfires.
