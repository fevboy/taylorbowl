Ephemera
For the commands that use Ephemera, see Invocations
Ephemera are an uncommon currency used to perform Invocations. They do not last forever; their name is derived from the word ephemeral, which means temporary. They are sometimes referred to colloquially as “ephys”.
Ephemera are obtained either through using credits or a random chance from Ephemera Level or IQ. While existing, they have a random chance to disappear. A user can only hold a limited number of Ephemera, dictated by their Ephemera Level.
Obtaining
Each Ephemera Level grants a 0.5% chance and each IQ grants a 0.005% chance at the end of every Bowl the user votes in to obtain an Ephemera, known as Spontaneous Generation. This includes Autovotes. These chances are independent, so there is a chance to obtain two Ephemera in a single Bowl.

You may also use the command !condense to convert credits into 1 Ephemera. Each user may only use !condense once per Bowl, which resets at the start of a Bowl. The price is determined by a number of factors, given by:

$$\text{Price}=\text{max}\begin{cases} 0.00075\cdot\text{Active credits} \\ 0.015\cdot\text{Total credits} \\ 1989 \end{cases}$$

This condensing limit can be superseded by the Invocation !hypercondense but it has a higher cost. If a user has reached their maximum allowed Ephemera, !condense will fail, and they cannot get spontaneous Ephemera.
Disintegration
Not to be confused with Disintegrate, an Evocation.
Every Ephemera has an independent 0.5% chance to Disintegrate at the end of every Bowl. This also means a user could lose more than one Ephemera at a time.

If a user has used Reinforce, they also have a 20% chance to have 1 Ephemera Disintegrate at the end of each Bowl. This chance is calculated as an additional chance after the standard Disintegration. Each Ephemera first has a 0.5% chance to Disintegrate. Then, if the user still has Ephemera, a single 20% chance is rolled for only one Ephemera to additionally Disintegrate.
Hypercondense
Hypercondense is an Invocation that can be used to create more Ephemera while bypassing the usual condensing limitations of 1 per Bowl, all at a higher cost. The command !hypercondense consumes all Ephemera, then creates double of what was consumed, with each new Ephemera costing 110% of the standard price.

The cost to Hypercondense is given by the formula:

$$\text{Cost}=2.2\cdot\text{Ephemera on-hand}$$

Hypercondense will misfire if the resultant Ephemera would exceed the invoker's Ephemera limit, or if the invoker has insufficient credits. However, it is unclear what the effects of this specific misfire are due to the involvement of credits.
Delegate
Delegate is an Invocation that can be used to transfer Ephemera from one user to another. The command !delegate @user can be used at the cost of 2 Ephemera to transfer all other Ephemera to the user specified.

If no user is specified, or there are no other Ephemera to delegate (i.e. the invoker only has 2 Ephemera), the command will misfire. If the Ephemera delegated would exceed the user's Ephemera limit, all excess Ephemera are lost.
An Ephemera Level is required to hold Ephemera. Each Ephemera Level allows the user to hold two Ephemera.

Each user starts with an Ephemera Level of 0. This can be raised by 1 using the command !exceed, which consumes credits. The price starts as 0.2% of active credits, or all credits owned by active users, and increases by an additional 0.2% for every Level higher. This is given by the formula:

$$\text{Price}=0.002\cdot\text{Active credits}$$

Ephemera Level has a limit of 20, so users can only hold up to a maximum of 40 Ephemera at one time, and have a maximum Spontaneous Ephemera chance from Ephemera Levels of 10%.

In the command !ephemera, the description of Invocations is hidden if the command user's Ephemera Level is too low. To view any given Invocation, the user's Ephemera Level must permit them to hold as many Ephemera as is required to use the Invocation. For example, !trick requires 10 Ephemera, so a user must have Ephemera Level 5. This is given by the formula:

$$\text{Minimum level}=\Biggl\lceil\cfrac{\text{Invocation cost}}{2}\Biggr\rceil$$

Each additional Ephemera Level also gives a user 40 more Risk and Anti-Risk rolls, and 100 more Paid Yahtzee rolls.

!collect: Uses 12 Ephemera to conduct one round of taxes. (Main article: Credits)

!restart: Uses 24 Ephemera to force a Bot Restart within 60 seconds. (Main article: Bot Cycle)

!abandon: Uses 28 Ephemera to remove all farmers' ability to harvest. (Main article: Farm)

!crystallize: Uses 32 Ephemera and 16 Ephemera Levels to create a Perpetua. (Main article: Perpetua)
As Spontaneous Generation and Disintegration are probabilistic, it is difficult to determine exactly how many Ephemera a user will have at a given time. However, it is possible to suggest averages.
Probabilities
The probability for an Ephemera to Spontaneously Generate is given by:

$$P(\text{EL generation})=0.005\cdot\text{EL} \\ P(\text{IQ generation})=0.00005\cdot\text{IQ}$$

The probability for an Ephemera to Disintegrate in a given Bowl is given by:

$$P(Y)=0.005$$

Generation
Let the event an Ephemera generates be \(X\). The probability for \(X=0\), \(X=1\), and \(X=2\) is given by:

$$P(X=0)=(1-0.005\cdot\text{EL})(1-0.00005\cdot\text{IQ}) \\ P(X=1)=0.005\cdot\text{EL}\cdot(1-0.00005\cdot\text{IQ})+0.00005\cdot\text{IQ}\cdot(1-0.005\cdot\text{EL}) \\ P(X=2)=0.0000005\cdot\text{EL}\cdot\text{IQ}$$

The expected value of Ephemera generation is given by:

$$E(X)=P(X=1)+2\cdot P(X=2)=0.005\cdot\text{EL}+0.00005\cdot\text{IQ}$$

Disintegration
Let the event an Ephemera disintegrates be \(Y\). Since each disintegration is independent, \(Y\) can be described with a binomial distribution:

$$Y\sim B(n,0.005)$$

where \(n\) is the number of Ephemera on-hand.

Thus, the probability that a certain number of Ephemera disintegrate in a given Bowl is given by:

$$P(Y=r)=\binom{n}{r}\cdot 0.005^{r}\cdot 0.995^{n-r}$$

where \(r\) is the number of Ephemera that disintegrate and \(n\) is the number of Ephemera on-hand.

And the expected value of Ephemera disintegration is given by:

$$E(Y)=0.005\cdot n$$

The disintegration of Ephemera can be described with an exponential decay function:

$$N(t)=N_{0}\cdot 0.995^{t}$$

where \(N(t)\) is a function of \(t\) for number of Ephemera on-hand, \(N_{0}\) is the original number of Ephemera on-hand, and \(t\) is the number of Bowls that have elapsed. Note that this formula assumes no new Ephemera generate.

Assuming no new Ephemera generate, the probability of having at least a certain number of Ephemera after a certain number of Bowls is given by the formula:

$$f(k,t)=\sum_{i=k}^{N_{0}} \cfrac{N_{0}!}{i!(N_{0}-i)!}\cdot 0.995^{ki}\cdot\left( 1-0.995^{k} \right)^{N_{0}-i}$$

where \(k\) is the ending number of Ephemera, \(t\) is the number of Bowls that have elapsed, and \(N_{0}\) is the original number of Ephemera.
Ephemera Equilibrium
Equilibrium Ephemera refers to the number of Ephemera a user will have on-hand, given they vote in every bowl for a sufficiently long time.

This value is reached when there is an equal probability for an Ephemera to Generate as there is for it to Disintegrate. When there are less Ephemera than Equilibrium, the probability for Generation is higher than that of Disintegration, and vice versa. Thus, equating the probabilities:

$$0.005\cdot\text{EL}+0.00005\cdot\text{IQ}=0.005\cdot N_{eq}$$

which simplifies to:

$$N_{eq}=\text{EL}+0.01\cdot\text{IQ}$$

where \(n_{eq}\) is your Ephemera Equilibrium.

The value given will likely not be an integer. However, it is not too significant as your Ephemera on-hand will naturally fluctuate.

The average number of Bowls needed to reach your Ephemera Equilibrium can also be calculated. The average number of Bowls required to gain 1 Ephemera is given by the formula:

$$t=\cfrac{1}{0.005\cdot\text{EL}+0.00005\cdot\text{IQ}-0.005\cdot{N_{0}}}$$

where \(t\) is the number of Bowls required and \(N_{0}\) is original number of Ephemera.

In other words, to go from \(N_{0}\) to \(N_{0}+1\) Ephemera, \(t\) Bowls are required. Thus, the number of Bowls required to go from 0 to your Ephemera Equilibrium is given by taking the summation from 0 to your Ephemera Equilibrium of the above:

$$t_{\text{total}}=\sum_{i=0}^{N_{eq}}\cfrac{1}{0.005\cdot\text{EL}+0.00005\cdot\text{IQ}-0.005\cdot i}$$

where \(N_{eq}\) is your Ephemera Equilibrium.

This can be generalised to start from and end on any number of Ephemera as such:

$$t_{\text{total}}=\sum_{i=N_{0}}^{N}\cfrac{1}{0.005\cdot\text{EL}+0.00005\cdot\text{IQ}-0.005\cdot i}$$

where \(N_{0}\) is your original Ephemera on-hand and \(N\) is your desired Ephemera on-hand.

You may use the EsteBot command /ephemeraequilibrium to determine your Ephemera Equilibrium, and the average number of Bowls required to reach a number of Ephemera below or equal to your Equilibrium.
Expected Ephemera
Expected Ephemera refers to the average number of Ephemera a user will Spontaneously receive given a certain number of Bowls voted in consecutively.

The formula derived earlier for the number of Bowls required to go from one number of Ephemera to another can be repurposed for this calculation, but to solve it is extremely difficult:

$$t=\sum_{i=N_{0}}^{N}\cfrac{1}{0.005\cdot\text{EL}+0.00005\cdot\text{IQ}-0.005\cdot i}$$

where \(t\) is the number of Bowls you can vote in consecutively and \(N\) is the number of Ephemera that you will end with.

The summation function cannot be simplified to solve for \(N\). It is far easier to solve it algorithmically, or by trial and error. Simply use /ephemeraequilibrium multiple times, modifying your desired number of Ephemera, and finding the largest value below the number of Bowls you wish to use.

You may use the EsteBot command /ephemeraexpected to estimate the average number of Ephemera that will Spontaneously Generate. However, it does not take into account Disintegration, and is thus more accurate when the number of Ephemera is small.
