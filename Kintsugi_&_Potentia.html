Kintsugi & Potentia
This page is under construction. We are currently looking for images to add to it. Please DM ev on Discord if you have any images that may be worthwhile. Thank you!
Kintsugi and Potentia are aspects of TaylorBowl that involve real money. They are bought using United States Dollars (USD). Both Kintsugi and Potentia can be obtained through Bianca's Kofi, accessible using the command !kofi. Kintsugi is a paid subscription service that increases credits obtained from voting in Bowls, grants access to an exclusive channel and role in the TaylorBowl Discord Server, and gives 2 Potentia on the 1st of every month. Potentia are a currency that can be converted to credits or Ephemera using commands. Besides Kintsugi, they can also be obtained through purchasing them directly.
Kintsugi is a subscription that gives subscribed users, also known as Kintsugi, benefits. Subscription renews monthly for $3 USD.
Benefits
Kintsugi receive a 2x multiplier to credits obtained from voting in Bowls. This multiplier can stack multiplicatively with the 1.25x Server Booster multiplier for a maximum of 2.5x. Kintsugi receive 2 Potentia on the 1st of every month, which are granted using the admin-only command !kintsugi. They are distributed in the Kintsugi channel. Kintsugi gain access to a special pale gold role with a symbol in the TaylorBowl Discord Server. In the user menu, they are displayed above Server Boosters but below the Council. This role also grants Kintsugi access to a Kintsugi channel.
Potentia are an uncommon currency that can be converted into credits or Ephemera. They are a permanent currency but only exist to be converted. They represent a potential increase in credits or Ephemera; their name is derived from the word potential. They are sometimes referred to colloquially as “potes”.
Obtaining
Potentia are obtained through purchasing using real-life currency. Each Potentia is bought using $1 USD through Bianca's Kofi page, which can be accessed using the command !kofi. Kintsugi also receive 2 Potentia on the 1st of every month.
Usage
Each Potentia can be converted into credits worth 0.05% of Active Credits using the command !summon number. Omission of the parameter number shows the number of Potentia of the command user, their total value in credits, and their individual value in credits. Each Potentia can also be converted into 1 Ephemera using the command !transfigure number. Omission of the parameter number shows the number of Potentia of the command user. Transfigured Ephemera can exceed the user's Ephemera limit by 2.
List of Commands
Command Description
!kintsugi Admin-only. Grants 2 Potentia to all Kintsugi.
!kofi Provides a link to Bianca's Kofi page.
!summon number Converts a specified number of Potentia into credits.
number is an optional parameter that refers to the number of Potentia converted, if omitted shows the command user's number of Potentia and the value of Potentia.
!transfigure number Converts a specified number of Potentia into Ephemera.
number is an optional parameter that refers to the number of Potentia converted, if omitted shows the command user's number of Potentia and the possible number of conversions.
